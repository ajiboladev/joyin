Frequently Asked Questions

What's the difference between the website and installed app? +
The installed version runs in its own window (like a native app), works offline, sends push notifications, and launches faster from your home screen.
Is it really free? No hidden costs? +
Yes, JOYIN is completely free. No subscriptions, no in-app purchases, no ads. We believe social connection should be accessible to everyone.
I don't see the install button. What should I do? +
Try these steps: 1) Use Chrome, Edge, or Safari 2) Visit the main JOYIN page first 3) Refresh the page 4) Look for the install icon in your address bar.
How do I uninstall JOYIN? +
On mobile: Press and hold the JOYIN icon, then tap "Uninstall" or "Remove from Home Screen". On desktop: Right-click the JOYIN window and select "Uninstall".
Will installing use storage on my device? +
Very little! JOYIN uses about 2-5MB of storage, compared to 100-200MB for typical social media apps. It's incredibly lightweight.
